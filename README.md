# QuantInsight
### Real-Time Market Probability Analytics Platform

QuantInsight is a data analytics project that explores how probability and statistical models behave using real-world financial market data. The platform integrates live market data through API connections and applies statistical analysis, probability distributions, and simulations to understand market dynamics.

---

## Project Objective

The goal of QuantInsight is to demonstrate the practical application of probability theory and statistics using real financial market data. Instead of relying on static datasets, the platform retrieves live market data through APIs and analyzes market behavior through statistical and probabilistic models.

---

## Key Features

- Live financial market data integration using APIs  
- Statistical analysis of market returns  
- Implementation of probability distributions  
- Monte Carlo simulation for market scenarios  
- Web-based visualization dashboard  

---

## Statistical Analysis

The system calculates core statistical metrics used in quantitative analysis:

- Mean
- Variance
- Standard Deviation
- Covariance
- Correlation

These metrics help analyze market volatility and relationships between assets.

---

## Probability Models Implemented

QuantInsight demonstrates multiple probability distributions applied to financial market behavior:

**Binomial Distribution**  
Probability of market moving up a certain number of times within a given number of trading days.

**Poisson Distribution**  
Used to model the frequency of rare events such as large price movements.

**Geometric Distribution**  
Used to estimate the waiting time until a particular market event occurs.

**Uniform Distribution**  
Used to model events where outcomes are assumed to be equally likely within a range.

---

## Monte Carlo Simulation

Monte Carlo simulations are used to generate multiple potential future market scenarios based on historical statistics. The simulation estimates probabilities of different price movements and helps visualize possible future outcomes.
